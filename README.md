## 💻 Developer Setup & Running Locally

If you'd like to run this 3D interactive portfolio locally, follow these steps:

### Prerequisites
- Node.js (v18.x or later recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/suyash-agrawal-mntta/suyash-agrawal-mntta.github.io.git
   cd suyash-agrawal-mntta.github.io
   ```

2. **Install the dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root directory and add your Gemini API Key:
   ```env
   NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to experience the 3D interaction and chat widget!

5. **Build for Production:**
   ```bash
   npm run build
   ```
